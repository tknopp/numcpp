Detailed Description

Graphics module containing rudimental graphics output functions.

The graphics module uses the cairo graphics library to generate simple graphs or images. The module is highly experimental. It requires to link against the libcairo.

Export the matrix x as pdf.

For converting of floating point values onto a colormap, one has to give the window parameters(winMin, winMax) and the colormap cm. In this overload of the function export_pdf, the parameters winMin and winMax are the minimum and maximum value of x. The default colormap is colormaps::gray.

This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts.
